Secret Bache (Floral Store)

Secret Bache is a beautifully designed flower shop website created to provide customers with an elegant and user-friendly online experience for exploring floral arrangements and decorative themes for different occasions. The website offers a wide variety of fresh flowers, bouquet collections, and creative decoration ideas suitable for weddings, birthday parties, corporate meetings, anniversaries, and other special events.

It also provides customized floral themes and decoration services for event planning, helping customers choose the perfect floral setup according to their needs and preferences. From romantic wedding decorations to professional corporate event arrangements, the platform presents multiple design inspirations and bouquet styles.

The website includes interactive features such as product browsing, flower selection, purchasing options, customer feedback submission, and easy navigation to ensure a smooth shopping experience. Users can explore different categories, view floral designs, and select the best arrangements for their occasions.

Built using HTML, CSS, JavaScript, PHP, and backend technologies, Secret Bache combines visual elegance with functional performance, making it a complete floral store web project designed for both beauty and convenience.
